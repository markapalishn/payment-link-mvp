# MVP Платежная ссылка

Прототип многоразовой платежной ссылки с функционалом создания, редактирования и предпросмотра.

## Возможности

- ✅ Создание и редактирование платежных ссылок
- ✅ Предпросмотр платежной страницы в отдельной вкладке
- ✅ Копирование ссылок
- ✅ Генерация и скачивание QR-кода
- ✅ Управление настройками платежной ссылки (фиксированная/свободная сумма, дополнительные поля)

## Структура проекта

- `index.html` - основной интерфейс личного кабинета
- `preview.html` - страница предпросмотра платежной ссылки
- `Fonts/` - шрифты T-Sans
- `t-business.png` - логотип

## Демо

Проект опубликован на GitHub Pages: https://markapalishn.github.io/payment-link-mvp/

## Использование

1. Откройте сайт в браузере или локально запустите `index.html`
2. Создайте новую платежную ссылку через кнопку "Создать ссылку"
3. Настройте параметры ссылки
4. Используйте кнопку "Предпросмотр" для просмотра страницы оплаты
5. Скачайте QR-код через меню копирования
